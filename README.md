## ReactEats - Food Delivering Webapp

### Deployment on vercel: https://react-eats.vercel.app/

### Technologies Used

- Front-end:
  `React`
  `Redux`
  `Tailwind CSS`
  `JavaScript`
  `React Router`

- API Integration : `Swiggy API`

## Setup the project

1. Fork the project
2. Clone project
3. Install dependencies `npm install`
4. To run the server execute `npm start`
